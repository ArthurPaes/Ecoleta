## Installation

### Clone

- Clone this repo to your local machine using:
```shell
$ git clone https://github.com/ArthurPaes/Ecoleta.git
```
### Setup
> Enter the folder:
```shell
$ cd Ecoleta
```
> Execute `yarn` or `npm instal` to install all the dependencies

```shell
$ yarn 
$ npm install
```
> Start the project:
```shell
$ yarn start
```
